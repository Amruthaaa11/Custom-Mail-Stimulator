# Custom-Mail-Stimulator
A playful Python script that simulates sending custom emails to a list of recipients. Each email is printed one at a time with a dramatic pause and screen clear between messages. Includes an interactive menu to manage the email list.

## Features

- Add or remove email addresses interactively
- View the current list of emails
- Simulate sending personalized messages to the first 10 recipients
- Clears the screen between each email for dramatic effect

## How It Works

- The script maintains a list of email addresses.
- You can add or remove emails using the menu.
- When you choose to "Get SPAMMING", the script prints a humorous custom message to each of the first 10 emails.
- After each message, the screen is cleared and the next email is printed.

## How to Run

### Prerequisites

- Python 3.x installed
- VS Code or any Python-compatible IDE
